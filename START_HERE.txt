🚀 3-MINUTE QUICK START

  Step 1: Run Database Migration (2 minutes)
  ─────────────────────────────────────────
  • Open: http://localhost/phpmyadmin
  • Select: ecommerce_marketplace database
  • Go to: SQL tab
  • Paste from: database_multi_image.sql
  • Click: Execute
  
  Step 2: Test Feature (1 minute)
  ───────────────────────────────
  • Visit: http://localhost/ecommerce/seller/dashboard.php
  • Click: "My Products" → "Add New Product"
  • You'll see TWO image upload fields:
    - "Product Image" (primary image)
    - "Add More Images" (select multiple)
  • Fill form, select multiple images, click "Add Product"
  
  Step 3: View Gallery
  ──────────────────
  • Click on the product you created
  • See beautiful gallery with main image + thumbnails
  • Click any thumbnail to switch the main image
  • Enjoy the smooth, professional gallery! 📸
